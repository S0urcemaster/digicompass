# Digi Compass Specification

Die bisherige Spezifikation wurde archiviert unter `/_spec/prev/2026-05-12-pre-rewrite/`.

## Eroeffnung

Produktnah:

Digi Compass hilft Menschen, aus bedeutungsvollen Elementen eine persoenliche Orientierung aufzubauen.

Technisch:

Digi Compass ist ein System zum Aufbau persoenlicher Orientierung aus bedeutungstragenden Elementen.

Diese Orientierung entsteht durch Auswaehlen, Einordnen, Bewerten und Verbinden von Elementen zu groesseren Einheiten.
