# Default Theme

Diese Datei beschreibt die Default-Richtung fuer die Darstellung von Digi Compass.

Sie ist absichtlich leicht und empfehlungsnah gehalten.

## Leitfrage

Welche UI-Richtung soll ein Agent zuerst versuchen, wenn keine engere Darstellungs-Schaerfung vorliegt?

## Rolle

Dieses Theme ist die erste Default-Empfehlung fuer die UI-Ableitung.

Es beschreibt keine starre Endform.

Es hilft einem Agenten, eine erste klare und zusammenhaengende Darstellung zu bilden, ohne die UI frueh zu stark festzulegen.

## Default-Empfehlungen

- Die App wirkt als zusammenhaengendes, ruhiges Hochformat-Objekt.
- Die Benutzerfuehrung bleibt auf wenigen klar lesbaren Ebenen.
- Eine aktuelle Einheit soll sichtbar hervortreten, ohne die naechsten moeglichen Schritte zu verstecken.
- Navigation, Auswahl und Bewertung sollen als zusammengehoerige Bearbeitung lesbar bleiben.
- Die UI darf einfach sein, soll aber nicht leer wirken.
- Wiederholung gleicher Muster ist hilfreicher als viele Sonderfaelle.
- Schaltflaechen auf dem App-Gehaeuse und benutzbare Elemente auf Images bleiben klar als unterschiedliche Arten lesbar.

## Nicht-Ziel

Diese Datei beschreibt absichtlich noch keine enge Komponentenform.

Sie legt insbesondere noch nicht fest:

- einen bestimmten Browser-Typ
- eine bestimmte Kartenlogik
- eine bestimmte Navigationsanordnung
- eine bestimmte visuelle Stilrichtung

## Offene Entscheidungen

- Welche konkrete Oberflaechenstruktur sich als staerkste Default-Richtung erweist, bleibt offen.
- Welche Navigationsmuster fuer Focus- und Mindset-Bearbeitung am lesbarsten sind, bleibt offen.
