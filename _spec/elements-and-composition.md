# Elements And Composition

Diese Datei beschreibt die Grundformen, aus denen Digi Compass aufgebaut ist.

Sie klaert, was ein Element ist, was eine Komposition ist und wie groessere Einheiten aus diesen Grundformen hervorgehen.

## Leitfrage

Aus welchen Grundformen ist Digi Compass aufgebaut, und wie gehen diese Grundformen in groessere Einheiten ueber?

## Grundformen

Digi Compass arbeitet mit zwei fachlichen Grundformen:

1. Element
2. Komposition

Diese beiden Grundformen tragen die fachliche Struktur des Produkts.

## Element

Ein Element ist die kleinste eigenstaendige Einheit von Digi Compass.

Ein Element kann:

- identifiziert werden
- angezeigt werden
- eingeordnet werden
- bewertet werden
- in groessere Einheiten eingehen

## Element-Struktur

Jedes Element traegt mindestens:

- eine stabile Identitaet
- eine bestimmbare Art
- einen darstellbaren Inhalt
- eine einordenbare Bedeutung
- eine bewertbare Relevanz

## Element-Arten

Die aktuelle Fachlogik zeigt mindestens diese Element-Arten:

- Saying
- Image

Weitere Element-Arten koennen hinzukommen, wenn sie dieselbe Grundform tragen.

## Komposition

Eine Komposition verbindet mehrere Einheiten zu einer neuen eigenstaendigen Einheit.

Diese Einheiten koennen Elemente oder bereits gebildete Kompositionen sein.

Eine Komposition kann:

- identifiziert werden
- angezeigt werden
- eingeordnet werden
- bewertet werden
- selbst wieder in groessere Einheiten eingehen

## Kompositions-Struktur

Jede Komposition traegt mindestens:

- eine stabile Identitaet
- eine bestimmbare Art
- eine Menge oder Ordnung ihrer Bestandteile
- eine darstellbare Gesamtform
- eine einordenbare Bedeutung
- eine bewertbare Relevanz

## Zusammensetzung

Eine Komposition kann auf zwei Arten entstehen:

1. aus Elementen
2. aus Kompositionen

Dadurch bleibt der Aufbau des Produkts ueber alle Ebenen hinweg aus derselben Logik gebildet.

## Aktuelle Einordnung

Die bisherige Produktstruktur laesst sich so einordnen:

- Saying ist ein Element.
- Image ist ein Element.
- Focus ist eine Komposition aus Elementen.
- Mindset ist eine Komposition aus Kompositionen.

## Rolle von Focus

Ein Focus verbindet bedeutungstragende Elemente zu einer gemeinsam wahrnehmbaren Einheit.

Ein Focus bildet aktuell die erste Kompositionsebene des Produkts.

## Rolle von Mindset

Ein Mindset verbindet mehrere Foci zu einer hoeheren Orientierungseinheit.

Ein Mindset bildet aktuell die zweite Kompositionsebene des Produkts.

## Gemeinsame Wirkung auf die Fach-Spezifikation

Alle weiteren Fachdateien bauen auf dieser Einteilung auf.

Das bedeutet:

- Datenmodelle beschreiben Elemente, Kompositionen oder ihre Beziehungen.
- Interaktionen wirken auf Elemente, Kompositionen oder ihre Zusammenstellung.
- Darstellungen zeigen Elemente, Kompositionen oder ihre Einordnung.
- Neue Produkteinheiten werden zuerst als Element-Art oder Kompositions-Art geklaert.

## Leitfragen fuer neue Produkteinheiten

Bei jeder neuen Produkteinheit soll zuerst klar werden:

- Ist sie ein Element oder eine Komposition?
- Welche Bestandteile traegt sie?
- Welche Bedeutung traegt sie als eigenstaendige Einheit?
- Wie geht sie in die naechste groessere Einheit ein?
